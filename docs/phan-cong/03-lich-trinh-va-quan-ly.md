# 7. Lịch làm việc 3 tuần

## Tuần 1 - Khởi tạo Khung (Boilerplate)
- **Ngày 1-2**: Thành lập khung repo (`backend/` và `extension/`).
- **Ngày 3-5**: TV5, TV6 làm mẫu extension đơn giản gửi text rác lên FastAPI. Các TV khác (2,3,4) xây dựng script test pipeline AI riêng lẻ.
- **Ngày 6-7**: Có bản demo Extension nối với API (chưa có AI thực, chỉ trả kết quả cứng để test UI).

## Tuần 2 - Tích hợp AI
- Ghép pipeline CLIP và Rule-based vào `backend/main.py`.
- Tối ưu hóa UI/UX của Extension để nhận kết quả phân tích và render mượt mà.
- Bắt đầu test 30 cases bằng cách dùng Extension duyệt qua các trang báo, mạng xã hội mẫu.

## Tuần 3 - Hoàn thiện
- Test cuối cùng trên các trang mạng thực tế.
- Chụp ảnh, quay video demo hệ thống (bôi đen text, click chuột phải/quét ảnh bằng extension).
- Viết README chi tiết (hướng dẫn tải Extension, chạy Backend).
- Viết báo cáo cuối kỳ.
